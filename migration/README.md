# 제품표준서 Excel to CSV 마이그레이션 도구

## 개요
이 도구는 한국 화장품 회사의 제품표준서 Excel 파일(.xls)을 분석하여 데이터베이스 임포트에 적합한 CSV 형식으로 일괄 변환해주는 도구입니다. 약 1,000개 이상의 파일을 10여 초 내에 처리할 수 있는 높은 성능과 정확도를 제공합니다.

### 주요 기능
- **데이터 자동 추출**: 제품 기본 정보, BOM(원료 구성), QC 규격, 개정 이력 등을 자동으로 분류하여 추출합니다.
- **알러젠(Allergen) 감지**: 시트 내의 복잡한 위치에서도 알러젠 정보를 지능적으로 찾아냅니다.
- **중복 관리**: 동일한 제품 코드가 여러 파일에서 발견될 경우, 이를 별도의 파일로 분리하여 데이터 무결성을 유지합니다.
- **대량 처리**: 약 1,000개의 파일을 처리하는 데 약 11초가 소요됩니다.

## 빠른 시작

### 1. 준비사항
- **Python 3.8 이상**: 시스템에 Python이 설치되어 있어야 합니다.
- **Excel 파일**: 변환할 제품표준서 파일(.xls)들을 특정 폴더에 모아두세요.

### 2. 설치 및 설정
1. 이 `migration/` 폴더를 원하는 위치에 복사합니다.
2. `config.example.py` 파일을 `config.py`로 복사합니다.
3. `config.py` 파일을 메모장 등으로 열어 `SOURCE_DIR` 경로를 실제 Excel 파일이 있는 경로로 수정합니다.

### 3. 실행
`run.bat` 파일을 더블 클릭하여 실행합니다. 필요한 라이브러리가 자동으로 설치되고 변환 작업이 시작됩니다.

### 4. 결과 확인
작업이 완료되면 `csv_output/` 폴더 내에 다음과 같은 파일들이 생성됩니다:
- `products.csv`: 제품 기본 정보
- `bom.csv`: 원료 구성 정보
- `qc_specs.csv`: 품질 관리 규격
- `revisions.csv`: 개정 내역
- (중복 데이터가 있을 경우) `duplicates_...` 파일들
